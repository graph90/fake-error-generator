# Fake Error Generator

A lightweight Kivy app that helps users craft polite, randomized error messages when they want to pause conversations. Provides severity levels (Soft / Medium / Hard) so you can gently delay responses without hurting anyone’s feelings. Open source, ad-free, and privacy-respecting — the user stays in control of every message.

## Download

Get the latest APK from the [Releases](https://github.com/graph90/fake-error-generator/releases/tag/MessageDeferral) page and install it on your Android device.

## Features

- Polite, randomized error messages  
- Adjustable severity levels: Soft / Medium / Hard  
- Open source and privacy-respecting  
- Lightweight and easy to use  

## Installation

1. Download the APK from the [Releases](https://github.com/graph90/fake-error-generator/releases/tag/MessageDeferral) page.  
2. Enable installation from unknown sources on your Android device.  
3. Open the APK and install.
